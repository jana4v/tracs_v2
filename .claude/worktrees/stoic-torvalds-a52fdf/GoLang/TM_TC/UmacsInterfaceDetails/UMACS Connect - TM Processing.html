    UMACS Connect¶

    Welcome to the UConnect API!

    This API allows you to interact with the UMACS (Unified Mission Analytics and Commanding Software) for spacecraft operations. UConnect provides endpoints for accessing spacecraft data, telemetry streams, equipment and more.

    API Endpoints¶

    Common Error Response Format¶

    For all REST API endpoints, the error responses follow a standard structure to provide consistency across the API. Below are the details of the error response format for different status codes.

    Error Response:

    {
    "status": "error",
    "error": {
        "code": "<error_code>",
        "message": "<error_message>"
    }
}

    Get List of Spacecraft IDs¶

    Description: Retrieves a list of all spacecraft IDs available in the UMACS system.

    GET http://10.22.14.188/spacecrafts

    Response:

    {
    "spacecrafts": ["sc001", "sc002", "sc003"]
}

    Notes:

    • Use this endpoint to get a list of all spacecraft IDs that can be used in subsequent API calls.

    Get Default Spacecraft ID¶

    Description: Retrieves the default spacecraft ID configured in the UMACS system.

    GET http://10.22.14.188/spacecrafts/default

    Response:

    {
    "default_spacecraft_id": "sc001"
}

    Notes:

    • Use this endpoint to determine the default spacecraft ID for operations.

    Get List of Streams for Spacecraft ID¶

    Description: Retrieves a list of telemetry streams for a given spacecraft ID.

    GET http://10.22.14.188/streams?spacecraft_id=sc001

    Response:

    {
    "spacecraft_id": "sc001",
    "streams": ["stream1", "stream2", "stream3"]
}

    Notes:

    • The spacecraft_id parameter is required.
    • Use this endpoint to get telemetry stream IDs associated with a specific spacecraft.

    Get List of Telemetry Formats for Spacecraft ID¶

    Description: Retrieves a list of telemetry formats for a given spacecraft ID.

    GET http://10.22.14.188/telemetry_formats?spacecraft_id=sc001

    Response:

    {
    "spacecraft_id": "sc001",
    "telemetry_formats": ["format1", "format2", "format3"]
}

    Notes:

    • The spacecraft_id parameter is required.
    • Use this endpoint to get available telemetry formats for a specific spacecraft.

    Get List of Pids for Telemetry Format and Spacecraft ID¶

    Description: Retrieves a list of parameter IDs (PIDs) for a given telemetry format and spacecraft ID.

    GET http://10.22.14.188/pids?spacecraft_id=sc001&telemetry_format=format1

    Response:

    {
    "spacecraft_id": "sc001",
    "telemetry_format": "format1",
    "pids": ["pid1", "pid2", "pid3"]
}

    Notes:

    • The spacecraft_id and telemetry_format parameters are required.
    • Use this endpoint to get PIDs associated with a specific telemetry format for a spacecraft.

    Get List of Values for Pids of Telemetry Format and Spacecraft ID¶

    Description: Retrieves the current telemetry values for a list of PIDs for a given telemetry format and spacecraft ID.

    POST http://10.22.14.188/values
Content-Type: application/json

    Request Payload:

    {
    "spacecraft_id": "sc001",
    "telemetry_format": "format1",
    "pids": ["pid1", "pid2", "pid3"]
}

    Response:

    {
    "spacecraft_id": "sc001",
    "telemetry_format": "format1",
    "frame_time": "11-DEC-2023 08:42:05:633",
    "pid_values": [
        {
            "pid": "PWRS0090",
            "tm_cnt": 219,
            "float_v": 36.57598,
            "str_v": "36.58"
        },
        {
            "pid": "DTG10001",
            "tm_cnt": 0,
            "float_v": 0,
            "str_v": "OFF"
        }
    ]
}

    Notes:

    • The spacecraft_id, telemetry_format, and pids parameters are required.
    • Use this endpoint to get the current values of specific PIDs for a telemetry format and spacecraft.
