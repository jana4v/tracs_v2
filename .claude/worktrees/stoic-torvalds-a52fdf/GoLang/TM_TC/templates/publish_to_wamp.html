Publish to WAMP Endpoint Documentation

POST /publish_to_wamp

Publishes a message to the specified WAMP topic.

Request

        Headers:
            Content-Type: application/json
        Body (for WampMessage):
        {
            "topic": "example.topic",
            "msg": {
                "key1": "value1",
                "key2": "value2"
            }
        }
        Body (for WampMessage1):
        {
            "topic": "example.topic",
            "msg": "string message"
        }

Response

        Headers:
            Content-Type: text/plain
        Body:
            Message published successfully

Example Request

        curl -X POST \
        http://localhost:8080/publish_to_wamp \
        -H 'Content-Type: application/json' \
        -d '{
              "topic": "example.topic",
              "msg": {
                "key1": "value1",
                "key2": "value2"
              }
            }'

Example Response

        Message published successfully

Error Responses

        Status Code: 400 Bad Request
        Body:
        {
            "error": "error message"
        }
        
        Status Code: 500 Internal Server Error
        Body:
        {
            "error": "Failed to publish message"
        }
