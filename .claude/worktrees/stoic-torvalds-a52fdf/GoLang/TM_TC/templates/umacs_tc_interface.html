POST /umacs_tc

Handles the TC request.

Request

        Headers:
            Content-Type: application/json
        Body:
        {
            "action": "loadprocedure",
            "proc_name": "exampleProcName"
        }

Response

        Headers:
            Content-Type: application/json
        Body:
        {
            "ack": true,
            "exe_status": "success"
        }

Example Request

        curl -X POST \
        http://localhost:8080/umacs_tc \
        -H 'Content-Type: application/json' \
        -d '{
              "action": "loadprocedure",
              "proc_name": "exampleProcName"
            }'

Example Response

        {
            "ack": true,
            "exe_status": "success"
        }
