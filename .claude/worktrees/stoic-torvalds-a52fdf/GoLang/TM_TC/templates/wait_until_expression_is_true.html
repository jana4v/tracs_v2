Wait Until Expression Is True Endpoint Documentation

POST /wait_until_expression_is_true

Evaluates the provided logical expression with the given variables and waits until the expression is true or the specified timeout is reached.

Request

        Headers:
            Content-Type: application/json
        Body:
        {
            "expression": "a > b",
            "variables": ["a", "b"],
            "timeout": 30
        }

Response

        Headers:
            Content-Type: application/json
        Body:
        {
            "result": true,
            "conditions": {
                "a": {
                    "condition": "a == 10",
                    "status": true,
                    "value": "10"
                },
                "b": {
                    "condition": "b == 5",
                    "status": true,
                    "value": "5"
                }
            }
        }
