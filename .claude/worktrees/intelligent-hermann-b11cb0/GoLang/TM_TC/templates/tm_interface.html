TM Interface Documentation

Endpoints

  • POST /inject_dtm
  • POST /get_tm

POST /inject_dtm

Injects telemetry data into Redis.

Request

        Headers:
            Content-Type: application/json
        Body:
        {
            "TM_MNEMONIC": 22
        }
        OR
        {
            "TM_MNEMONIC": {
                "param_id": "PW1234",
                "param": "mnemonic",
                "source_info": "dtm",
                "raw_count": 10,
                "proc_value": "value",
                "time_stamp": "00:22:12",
                "upper_limit": 1,
                "lower_limit": 0,
                "err_desc": ""
            }
        }

Response

        Headers:
            Content-Type: text/plain
        Body:
            OK

Example Request

        curl -X POST \
        http://localhost:8080/inject_dtm \
        -H 'Content-Type: application/json' \
        -d '{
              "TM_MNEMONIC": 22
            }'

Example Response

        OK

POST /get_tm

Fetches telemetry data from Redis based on mnemonics and source.

Request

        Headers:
            Content-Type: application/json
        Body:
        {
            "mnemonics": ["key1", "key2"],
            "source": "TM1"
        }

Response

        Headers:
            Content-Type: application/json
        Body:
        {
            "results": {
                "key1": 123.45,
                "key2": "some_string"
            }
        }

Example Request

        curl -X POST \
        http://localhost:8080/get_tm \
        -H 'Content-Type: application/json' \
        -d '{
              "mnemonics": ["key1", "key2"],
              "source": "TM1"
            }'

Example Response

        {
            "results": {
                "key1": 123.45,
                "key2": "some_string"
            }
        }

Error Responses

        Status Code: 400 Bad Request
        Body:
        {
            "error": "Invalid request payload"
        }

        Status Code: 500 Internal Server Error
        Body:
        {
            "error": "Failed to process request"
        }
